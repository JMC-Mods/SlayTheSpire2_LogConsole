[h1]Log Console[/h1]
[h2] 1. Overview [/h2]
Strange issues in the game? MOD stopped working? Feature behaving incorrectly? With this MOD, you can conveniently view logs to help locate errors.

[url=https://github.com/JMC-Mods/SlayTheSpire2_LogConsole] GitHub Repository [/url]

[h2] 2. Features [/h2]
[list]
    [*] Displays a log window for viewing logs in game
    [img]https://images.steamusercontent.com/ugc/11293785759077049187/7A894D00F2B92D6E18DDE28741949292C7A9E312/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] The log window can be opened or closed with a hotkey (configurable, with controller and key combination support), and the configuration can be changed in the settings
    [img]https://images.steamusercontent.com/ugc/12334606906893322475/BDCB903CDC2821D5E139A21BA68E10439DA8D11B/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Adjusts the minimum displayed level to filter the log levels you care about
    [img]https://images.steamusercontent.com/ugc/16498039244173584933/EEE0AAEB8085A28804241615C4F0613B38F39EEA/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] When using multiple monitors, you can configure which monitor is used by default
    [img]https://images.steamusercontent.com/ugc/11470271167510541294/5C235D1BABCFC3B780DA430737EA407D23550162/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Supports regex filtering
    [img]https://images.steamusercontent.com/ugc/9945922665095272765/165A80E5DD78FEB73B0FBB30BB17E230A2706900/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
[/list]

[h2] 3. Notes [/h2]
[list]
    [*] Because of the game's mechanics, installing any MOD for the first time can cause save desynchronization. You can use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747533700] BetterSaveSlots [/url] to solve this
    [*] This MOD requires the following dependency:  
• [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747526103] JmcModLib [/url]  
    [*] The localization content of this MOD (including the workshop page and changelog), except Chinese, is generated by ChatGPT. It currently supports all languages officially supported by the game. If you think any wording is inaccurate or would like to contribute your own translation, feel free to open an issue / PR on GitHub or contact me directly
[/list]

[h2] 4. Compatibility [/h2]
[list]
    [*] This mod makes extensive use of reflection and may stop working after game updates. If you find that the mod no longer works, please report it promptly.
[/list]
