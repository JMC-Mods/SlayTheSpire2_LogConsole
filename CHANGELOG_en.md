# 🧾 Changelog

All notable changes to this project will be documented in this file.

Versioning scheme: Major.Minor.Patch. Major versions are used for substantial feature improvements; minor versions are generally incremented after a new version is published on the Steam Workshop; patch versions are incremented after each commit involving code changes (starting from 0).

## [1.3.0] - 2026-7-11
### Added
- Added an “Export Diagnostics” feature for exporting a `getlogs` diagnostic package.

## [1.2.0] - 2026-6-19
### Changed
- Updated the project to the official MOD release format.

## [1.1.0] - 2026-5-15
### Added
Initial release

## [1.0.0] - 2026-04-29
### Added
- Initially created the log console project, providing in-game log capture, a separate log window, cache clearing, and basic filtering.
